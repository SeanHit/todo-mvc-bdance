## 主要使用的技术
1. 单页面应用
2. 使用了三种不同的状态管理
   1. state
   2. hooks
   3. redux
3. 本地数据管理 localStorage本地管理
4. 使用node.js，express,mongodb搭建简单的服务端框架
5. nginx搭建前端项目
6. 个人服务器搭建相关技术，使用pm2将后端代码部署到云服务器上
   1. 实现基本的用户登录注册相关功能
   2. 实现简单的用户操作后台数据的动作
   3. 使用本地在cookie中存储了用户id
6. 由于种种原因，还没有做到对前端数据的收集处理，这个是下一步的工作方向
